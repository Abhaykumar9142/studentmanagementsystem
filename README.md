# Student Management System (Java)

##  Description

A console-based Student Management System developed using Core Java.
This application allows users to manage student records efficiently through a menu-driven interface.

The project is designed to strengthen understanding of **Object-Oriented Programming (OOP)** concepts and **Java Collections**.

---

##  Features

* Add new student records
* View all students
* Search student by roll number (supports alphanumeric roll numbers)
* Delete student record
* Menu-driven program for easy interaction

---

##  Technologies Used

* Java (Core Java)
* Object-Oriented Programming (OOP)
* Java Collections Framework (ArrayList)
* Scanner class for input handling

---

##  Project Structure

```
StudentManagementApp.java
```

*(Single-file project containing all required classes)*

---

##  How to Run the Project

1. Clone the repository:

   ```
   git clone https://github.com/yourusername/student-management-system.git
   ```
2. Navigate to the project folder
3. Compile the program:

   ```
   javac StudentManagementApp.java
   ```
4. Run the program:

   ```
   java StudentManagementApp
   ```
